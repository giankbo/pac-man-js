# pac-man-js
PAC-MAN can move freely by using the keyboard inputs. However, it can't go off the edge. It can't escape!

## what do we see here?
We mainly see here:

- PAC-MAN drawn in JS using `arc()` method.
- Detecting screen edge.
- Keyboard input for movements.
